Test du composant Inventaire

Pour tester le composant inventaire :

  1. Ouvrez index.html dans votre navigateur
  2. Cliquez sur l'onglet "Inventaire" dans le menu de navigation
  3. Vérifiez que le composant affiche :
    • Le titre "Gestion de l'inventaire"
    • Les boutons "Réception" et "Nouvel article"
    • Les cartes de statistiques (Articles en stock, Alertes stock, Lots expirant, Valeur totale)
    • La sous-navigation avec les onglets "Articles", "Mouvements" et "Commandes"
    • Le tableau des articles avec les colonnes SKU, Nom, Stock, etc.
  4. Testez les interactions :
    • Cliquez sur "Nouvel article" pour voir le formulaire d'ajout
    • Cliquez sur "Réception" pour voir le formulaire de réception
    • Cliquez sur l'icône œil d'un article pour voir ses détails
    • Naviguez entre les onglets "Articles", "Mouvements" et "Commandes"

Fonctionnalités du composant Inventaire :

  • Gestion des articles : Liste, ajout, modification, visualisation détaillée
  • Suivi des stocks : Niveaux actuels, seuils d'alerte, statuts
  • Traçabilité : Numéros de lot, dates de péremption
  • Mouvements de stock : Entrées, sorties, ajustements avec historique
  • Réception de marchandises : Formulaire avec contrôles HACCP
  • Alertes visuelles : Stocks bas (rouge), expiration proche (orange/jaune)
  • Recherche et filtres : Par nom, catégorie, statut
